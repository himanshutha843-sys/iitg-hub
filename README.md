# IITG Student Hub

A full-stack student resource platform for IIT Guwahati students.

## Features
- Semester-wise resources
- Subject pages
- PYQs and notes
- Upload system
- MongoDB database integration
- React frontend
- Node.js backend

## Tech Stack
- React
- Tailwind CSS
- Node.js
- Express
- MongoDB

## Setup

### Frontend
```bash
npm install
npm run dev
```

### Backend
```bash
cd server
npm install
node server.js
```

### MongoDB
Run:
```bash
mongod
```
